DeltaSqlAstBuilder

DeltaSqlAstBuilder is a command builder for the Delta SQL commands described in DeltaSqlBase.g4 ANTLR grammar.

Table 1. DeltaSqlAstBuilder’s Visit Methods
Name Description

CONVERT TO DELTA

Creates a ConvertToDeltaCommand

DESCRIBE DETAIL

Creates a DescribeDeltaDetailCommand

DESCRIBE HISTORY

Creates a DescribeDeltaHistoryCommand

GENERATE

Creates a DeltaGenerateCommand

VACUUM

Creates a VacuumTableCommand

DeltaSqlParser is used exclusively by DeltaSqlParser.
