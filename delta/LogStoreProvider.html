LogStoreProvider

LogStoreProvider is an abstraction of providers of LogStores.

LogStoreProvider uses the spark.delta.logStore.class configuration property (default: HDFSLogStore) for the fully-qualified class name of the LogStore to create (for a DeltaLog, a DeltaHistoryManager, and DeltaFileOperations).

Creating LogStore — createLogStore Method

createLogStore(
  spark: SparkSession): LogStore
createLogStore(
  sparkConf: SparkConf,
  hadoopConf: Configuration): LogStore
